# 常见问题

{% folding blue::什么是挂刀？为什么要挂刀？怎么挂刀？ %}

简单地说，挂刀（倒余额）就是利用可交易游戏道具在 Steam 市场与第三方平台的差价，以一定折扣获取 Steam 钱包余额的操作。

{% folding blue::各种挂刀比例 / 挂刀指数是什么意思？ %}

● 所有挂刀比例都是税后比例，即：实付金额 / 钱包到手余额

● 所有挂刀比例均是越低越好

● 新手建议优先参考稳定求购比例，优先选择日成交量相对较高的饰品

● 挂刀指数仅是对市场的一种定量分析指标，并不代表此时挂刀的实际比例，但一般挂刀指数越低，挂刀的收益越高。

{% endfolding %}

{% folding blue::为什么挂刀能省钱？原理是什么？ %}

简单地说，由于现金与 Steam 钱包余额的兑换是单向的，因此同样数值的现金相较于 Steam 钱包余额天然有着更高的价值。

这类似于大部分游戏的点券：现金可以单向兑换点券，但点券无法直接兑换现金。如果想换回现金，则要通过第三方平台折价兑换。

考虑到 CSGO / Dota2 饰品可以在 Steam 市场与第三方平台自由流通，其价值保持恒定。因此，对于同样的饰品，以 Steam 钱包余额计价的价格应当相较以现金计价的价格更高。

因此，挂刀相当于利用现金，低价收购了其他玩家手中的余额。只要饰品价格差带来的收益超过了交易手续费成本（主要是 Steam 市场的 15%），则能以同样金额获得相较于直接充值更多的 Steam 余额。

# 前置准备

1. Steam 挂刀行情站 [iflow.work](https://www.iflow.work/)

2. 需开通Steam市场 请参考[Steam:交易与市场限制](https://help.steampowered.com/zh-cn/faqs/view/451E-96B3-D194-50FC)

3. 用到的第三方交易平台：悠悠有品、网易BUFF、EcoSteam、IGXE、C5等，并绑定Steam账户

# 挂刀教程

1. 打开行情站设置平台与价格区间，按最优寄售比例比例排序，挑选合适的饰品，将交易量设置为大于200。

![20260212182607](https://img.099115.xyz/img/20260212182607.png)

2. 确认平台售价与Steam求购价无误，平台下单，等待卖家发货。

3. 参考最低寄售价，设定价格，等待饰品在 Steam 市场售出，挂刀完成。

# 注意事项

1. 务必选择成交量大于200的饰品进行交易！

2. 饰品购买后存在约7天交易冷却期，期间无法上架Steam市场，请选择价格比较稳定的饰品。如下图：

![20260212183123](https://img.099115.xyz/img/20260212183123.png)

3. Steam市场交易将收取15%手续费，已在行情站中计算。
